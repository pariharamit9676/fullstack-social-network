# Social Media Web App

A full-stack social media application built using **React**, **Node.js**, and **MySQL**, featuring user authentication, post creation, image uploads, likes, comments, and real-time chat functionality using **Socket.io**.

## 🚀 Tech Stack

- **Frontend**: React.js, Tailwind CSS, Axios
- **Backend**: Node.js, Express.js, MySQL
- **Real-time**: Socket.io
- **Image Upload**: Multer (stored in public folder)
- **Authentication**: JWT

---

## ✨ Features

- 🔐 User Signup / Login
- 📝 Create and delete posts with images
- ❤️ Like and comment on posts
- 💬 Real-time chat between users
- 🧑‍💼 User profile with bio and profile picture
- 📁 Upload and serve images from local `public/` folder
- 🌐 Responsive design (mobile & desktop)
